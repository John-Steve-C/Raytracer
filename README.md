## 项目内容

利用 `Rust`实现一个 ray_tracer，追踪光线以绘制一张独特的图片。

[系列教程](https://raytracing.github.io/)

### 核心操作：

1. 计算从眼睛（原点）出发，到达像素的光线路径
2. 判断光线的交点/反射
3. 计算/设置交点的 RGB 颜色，得到图片
